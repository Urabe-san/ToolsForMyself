コマンドライン版ファイルマネージャーサンプル

■プログラム名
	FDop.exe

■引数
	FDop.exe [Option] [File/Directory Name] <Parameter>|<Distination>

■引数の詳細
	[Option]
		オプションの指定は大文字と小文字の区別をしません。
		create
			ファイルを作成します。
			作成したファイルのサイズはゼロバイトです。
			[File/Directory Name]に相対パス、または絶対パスでファイル名を指定する必要があります。
			ディレクトリの作成はできません。
			すでにファイルが存在する場合は失敗します。
			ファイルを作成するディレクトリには、ファイルを作成するために必要なアクセス権限が付与されている必要があります。
		delete
			ディレクトリまたはファイルを削除します。
			[File/Directory Name]に削除する対象を指定する必要があります。
			ディレクトリを削除する場合は、配下のファイルすべてが削除されます。
			配下のディレクトリまたはファイルを削除する権限がない場合は処理が途中で終了します。
			途中で処理が終了した場合も、削除したファイルは復活しません。
		copy
			ファイルをコピーします。
			ディレクトリのコピーはできません。
			[File/Directory Name]に相対パスまたは絶対パスでコピー元のファイル名を指定する必要があります。
			<Distination>にはコピー先のディレクトリを指定する必要があります。
			以下の場合はファイルのコピーが失敗します。
				コピー元のファイルにアクセス権限がないとき
				コピー元のファイルが存在しないとき
				コピー先のディレクトリに必要な権限がないとき
				コピー先のディレクトリ内にすでにファイルが存在するとき
				コピー先のディレクトリが存在しないとき
			コピー先のファイルが既に存在する場合に、上書きの確認はしません。
			コピー先のディレクトリが存在しないときにディレクトリが作成されることがありません。
			ファイルの属性の引継ぎはコマンドを実行したオペレーティングシステムのAPIに依存します。
			ファイルの所有権などのアクセス権限は実行したオペレーティングシステムのAPIに依存します。
		move
			"copy"を指定したときの仕様と同じですがコピーが完了したあとに、コピー元のファイルが削除されます。
			コピー元のファイルを削除する権限がないときは処理が失敗します。ただし、コピーが完了しているため結果は"copy"を指定した場合と同じです。
		rename
			ほとんどの動作は"move"と同じです。
			同じディレクトリ内でのファイル名変更となる場合にのみ、"move"とは違う挙動をします。
			ファイルにアクセスする権限がない場合は失敗します。
		attribute
			[File/Directory Name]に指定したディレクトリまたはファイルの属性を<Parameter>に指定した内容との通り変更します。
			[File/Directory Name]にディレクトリを指定したときの動作はコマンドを実行したオペレーティングシステムのAPIに依存します。
			(APIがエラーを例外とした場合は処理エラーとなりますが、単なるエラーとした場合はこのコマンドはエラーを返しません。)
		information
			次の通りの情報を表示します。
				ファイルかディレクトリであることの区別
				ファイルサイズ(ファイルの場合のみ)
				作成日付
				最終更新日時
				最終アクセス日時
				ファイルのフルパス
			表示内容は一行で表示されます。
			表示される情報はタブ文字で区切られます。
			[File/Directory Name]にディレクトリのフルパス、またはファイルのフルパスを指定する必要があります。
		list
			[File/Directory Name]にディレクトリのパスを指定する必要があります。
			[File/Directory Name]にファイルのパスを指定することはできません。
			指定されたディレクトリの配下のファイル一覧を表示します。
			表示の仕様はオプション"information"の表示内容と同じです。

	[File/Directory Name]
		ディレクトリまたはファイルを指定します。
		[Option]に指定したオプションによってはフルパスである必要があります。

	[Parameter]
		[Option]に"attribute"を指定した場合は必須です。
			+ or -: add or remove attribute
			指定の属性(空白文字で区切らず複数の指定をすることができます)
				a archive
				r read only
				h hidden
				s system

	<Distination>
		[Option]に"copy"、"move"、"rename"のいずれかを指定した場合は必須です。
